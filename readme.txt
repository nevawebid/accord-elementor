== Description ==

**ACF/SCF Accordion for Elementor** is a powerful WordPress plugin that seamlessly integrates ACF (Advanced Custom Fields) or SCF (Secure Custom Fields) repeater data with Elementor's page builder. Create stunning, responsive accordions with smooth animations and full customization options.

== Installation ==

### Automatic Installation

1. Log in to your WordPress admin dashboard
2. Navigate to Plugins ‚Üí Add New
3. Search for "ACF/SCF Accordion for Elementor"
4. Click "Install Now" and then "Activate"

### Manual Installation

1. Download the plugin zip file
2. Go to Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Choose the downloaded zip file and click "Install Now"
4. Activate the plugin after installation

### Setup Requirements

1. **Install Dependencies**:
   - Elementor (version 3.0 or higher)
   - ACF (Advanced Custom Fields) OR SCF (Secure Custom Fields)

2. **Create Custom Fields**:
   - Create a new repeater field in ACF or SCF
   - Add sub-fields for title and content within the repeater

3. **Use the Widget**:
   - Edit any page/post with Elementor
   - Search for "ACF/SCF Accordion" widget
   - Drag it to your desired location
   - Configure the settings in the widget panel

== Frequently Asked Questions ==

= Does this plugin work with both ACF and SCF? =

Yes! The plugin automatically detects which custom fields plugin you have installed (ACF or SCF) and adapts accordingly. You only need one of them installed.

= Can I use this plugin without Elementor? =

No, this plugin is specifically designed as an Elementor widget and requires Elementor to function.

= Is the accordion responsive? =

Absolutely! The accordion is built with a mobile-first approach and is fully responsive across all devices.

= Can I customize the icons? =

Yes, you can choose any icon from Elementor's icon library and set different icons for collapsed and expanded states.

= Does it support custom CSS? =

Yes, you can add custom CSS through Elementor's custom CSS feature or your theme's custom CSS section.

= Is it compatible with caching plugins? =

Yes, the plugin is compatible with all major caching plugins including WP Rocket, W3 Total Cache, and WP Super Cache.

= Can I use multiple accordions on the same page? =

Yes, you can add multiple accordion widgets to the same page, each with different data sources and settings.

= Does it work with custom post types? =

Yes, you can specify any post ID to pull ACF/SCF data from any post type.

= Is the plugin translation ready? =

Yes, the plugin includes a .pot file and is fully translation ready with proper internationalization.
